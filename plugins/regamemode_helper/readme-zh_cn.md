# Regamemode_Helper Plugin for MCDReforged
此插件重构自 [Gamemode-Helper](https://github.com/xksnetcbs/MCDR-gamemode_helper-1.16-)

安装插件前请确保您的MCDR大于等于2.0.0

# Command
| 指令 | 功能                                          |
|----|---------------------------------------------|
| !c | 更改您的模式并记录当前位置与朝向                            |
| !s | 更改您的模式为生存并传送回使用`!c`指令前的位置, 如果没使用过`!c`则无视此指令 |
